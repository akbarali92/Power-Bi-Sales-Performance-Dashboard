# Power-Bi-Sales-Performance-Dashboard
This project presents an interactive Sales Performance Dashboard built using Power BI. The dashboard analyzes retail sales data to track key performance indicators (KPIs), identify sales trends, and support data-driven business decisions. The project demonstrates the complete Power BI workflow from data import to visualization and insights.
Objectives

Monitor overall sales performance

Compare sales across regions, products, and categories

Identify top-performing products

Analyze customer payment behavior

Build an interactive and user-friendly dashboard
